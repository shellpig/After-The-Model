# After-The-Model — 主角設定

> 從原始 6 個候選身份（AI 善後員、記憶撿骨師、非法資料搬運工、真人代理、類比傾聽者、肉身資料農夫）+ 後續 3 個擴展候選（AI 申訴代辦、街頭表演者、城市觀察者）收斂後的最終方向。本檔為**已確認**設計，後續實作依此進行。完整候選腦力激盪可在 git 歷史中查回。

## 一句話定位

> **「白天替 AI 收拾殘局的善後員，業餘時拾起這個城市還沒被忘記的痕跡。」**

機械、冰冷、藍領的正職 × 對抗遺忘、人性、眼光的副業 — 主角形象由這兩條情緒線同時立體化。

---

## 主身份:AI 善後員(AI Cleaner / Maintenance)

### 概念

AI 雖然強大但偶爾會在物理世界出包:送貨無人機卡在死角、自動販賣機暴走、清潔機器人胡言亂語。主角拿著微薄薪水,專門替 AI 收拾實體殘局,是機器社會的底層勞工。

對應遊戲概念書「**替 AI 修正錯誤的人類校對員**」職業的物理版。

### 玩法核心

| 階段 | 玩家行為 |
|---|---|
| 接案 | 零工 App 派工,顯示故障 AI 位置 / 類型 / 報酬 |
| 移動 | 走路 / 地鐵前往任務地點 |
| 診斷 | 觀察故障 AI 行為、聽它胡言亂語、查附近線索 |
| 處理 | 修復 / 拆解 / 拔晶片 / 回收 |
| 交差 | 把成果交給回收商或上傳 App |

### 視覺定位

| 元素 | 說明 |
|---|---|
| 服裝 | 油污工裝、反光條工作背心 |
| 配備 | 工具腰帶、防護鏡、頭燈 |
| 氣質 | 藍領、疲憊、熟練、不愛說話 |
| 對比 | 跟富人區的 AI 管家 / 自動駕駛形成強烈階級反差 |

### 敘事潛力

> **「每集都有新型故障 AI」是天然的碎片化敘事載體。**

每個故障都是一則小故事 / 黑色幽默 / 反諷:

- 胡言亂語的清潔機器人在背誦上個世代的廣告
- 暴走販賣機只認某種已停產的代幣
- 卡死的送貨無人機懷裡抱著無法送達的包裹

## 副業:拾遺者(The Gleaner)

> 由原候選 #6「記憶撿骨師」(實體舊物)+ #9「城市觀察者」(抽象畫面)合併而成的副業。兩者本質都是「對抗遺忘」,合併後成為一個有重量的主題,而非兩個拼湊的小玩意。

### 概念

在 AI 抹平一切的城市裡,主角業餘時收集即將消失的「前 AI 時代」痕跡。這份收集分兩種媒材——**實體**(撿到的舊物)與**抽象**(拍到的畫面、錄到的聲音、寫下的觀察)——但本質都是**對抗遺忘**。

英文名 `The Gleaner` 取自 Agnes Varda 紀錄片《拾穗者與我》,原指收割後撿穀粒的人,引申為「收集被忽略事物的人」。

對應遊戲概念書「社會觀察」「碎片化故事」「都市漫遊感」「孤獨感」。

### 玩法的兩條分支

主角身上同時帶兩種工具,視情況使用:

| 媒材分支 | 工具 | 收集對象 |
|---|---|---|
| **實體**(撿) | 工具包、撬棒、手電筒、放大鏡 | 照片、卡帶、紙本、舊型記憶卡、小物件、信件 |
| **抽象**(記錄) | 老相機、筆記本、錄音筆 | 畫面、對話、聲音、街頭現象、AI 故障破綻 |

兩條線共用:

- 同一個公寓收藏空間(牆面 / 抽屜 / 筆記本)
- 同一群 NPC(懷舊收藏家、地下雜誌主編、研究機構、原主後人)
- 同一個敘事系統

### 抉擇軸(賣 / 留 / 還或發表)

每件收藏都可以選擇處置方式,影響副業劇情線與道德軸:

| 媒材 | 賣 | 留 | 還 / 發表 |
|---|---|---|---|
| 實體舊物 | 賣給黑市 / 收藏家 | 收進公寓 | 還給原主或其後人 |
| 影像紀錄 | 賣給地下雜誌 | 保留個人簿 | 上傳地下網路 |
| 聲音紀錄 | 賣給聲音檔案商 | 收進音檔庫 | 串流地下廣播 |
| 觀察筆記 | 賣給研究機構 | 收進日記 | 發表成文章 |

### 視覺定位

| 元素 | 說明 |
|---|---|
| 配備 | 手電筒、放大鏡、小袋子、老相機、筆記本、錄音筆 |
| 動作 | 蹲下翻找、撥開雜物、舉相機、停下來看、寫筆記 |
| 氛圍 | 跟主身份的「冷機械」對比,這條線是「**暖人味**」+「**眼光**」 |

### 敘事潛力

> **每件實體舊物 = 一個前時代的人類故事;每段抽象紀錄 = 一個正在消失的當下。**

實體:

- 一張舊照片 → 一段被取代的職業 / 一段失去的關係
- 一張舊型記憶卡 → 某人想保留但被系統強制刪除的記憶
- 一本紙本筆記 → 某個拒絕數位升級者的最後遺物

抽象:

- 一段街友哼的老歌 → 一個失傳的曲目
- 一張 AI 廣告故障畫面 → 一則諷刺
- 一段地鐵車廂的階級對比 → 一張城市切片

### 兩條線交織的化學反應

合併設定的真正價值——兩條線可以**互相觸發**,形成碎片化敘事的具體機制:

| 範例情境 | 流程 |
|---|---|
| **時光對照** | 撿到 30 年前的舊照片 → 在街上認出畫面中的位置 → 拍下「同一個地方今天的樣子」 → 兩張並列成為一個故事 |
| **拼湊陌生人** | 在地鐵拍下一張臉的瞬間 → 之後在主任務現場撿到一封寫給她的信 → 主角開始拼湊這個陌生人的人生 |
| **聲音遺跡** | 錄到街友哼一段老歌 → 在某個廢屋找到對應的舊卡帶 → 兩者匹配後解鎖原主的故事 |
| **意外見證** | 紀錄某 AI 故障畫面 → 後來該畫面變成主任務的證據鏈 |

## 主副交織(職業層級)

| 主 | 副 | 化學反應 |
|---|---|---|
| AI 善後員 | 拾遺者 | 處理故障 AI 時,**從它肚子裡掉出舊時代物件 / 拍下故障的諷刺畫面** → 兩條任務線天然交織 |
| 機械、冰冷、藍領 | 對抗遺忘、人性、眼光 | 兩種情緒對比,主角形象立刻立體 |
| 玩法:尋找 + 拆解 | 玩法:翻找 + 鑑定 + 觀察 + 記錄 | 動作機制可共用,新增的相機 / 筆記系統獨立 |
| 報酬:穩定但微薄 | 報酬:偶爾、不確定、可能很高 | **真實底層生存感** — 靠正職吃飯,副業是希望 |

## 與《遊戲概念》的對應

| 概念書元素 | 主角設定如何呼應 |
|---|---|
| 「替 AI 修正錯誤的人類校對員」 | ✅ AI 善後員 = 物理版 |
| 「低階層人類在 AI 時代的日常生活」 | ✅ 兩種職業都是底層、不酷、真實 |
| 「碎片化故事」 | ✅ 每個故障 AI、每件遺物、每段紀錄都是一個小故事 |
| 「都市漫遊感」 | ✅ 拾遺者天然鼓勵「停下來看、多走幾步、注意角落」 |
| 「社會觀察」 | ✅ 拾遺者的「抽象紀錄」這條線就是觀察的具體機制 |
| 「孤獨感」 | ✅ 善後員看遍 AI 出包,拾遺者看遍人被遺忘的痕跡 |
| 「不是英雄、不打倒邪惡企業」 | ✅ 主角只是想活下去 + 留下點什麼 |

## 任務範例(示意,MVP 範圍待 vertical slice 規格書決定)

**任務:地鐵站清潔機器人胡言亂語**

| 元素 | 內容 |
|---|---|
| 派工 | 街上 AI 善後員工會中介 / 零工 App push |
| 場景 | 一條街(接案)→ 地鐵站(處理)→ 小公寓(事後處理收藏) |
| NPC 1 | 派工方(中介人 / 工會櫃台) |
| NPC 2 | 回收商(拿晶片去交差) |
| 主流程 | 找到機器人 → 觀察胡言亂語線索 → 拆晶片 → 交回收商 |
| 拾遺機會(實體) | 從機器人記憶體裡撿出一張前時代舊照片 |
| 拾遺機會(抽象) | 拍下機器人胡言亂語的瞬間 / 錄下它背誦的過時廣告 |
| 副業抉擇 | 賣 / 留 / 還或發表 → 埋下副業劇情種子 |

**這個任務可同時驗證**:

- 主玩法(善後流程)
- 副玩法兩條線(撿 + 記錄)
- 三場景串接(街 / 地鐵 / 公寓)
- 兩個 NPC 對話
- 敘事選擇分支

MVP 階段要做哪些副業分支(實體 only / 抽象 only / 兩者皆做)留待 **vertical slice 規格書**決定。

## 未來擴展方向(暫定,非 MVP)

| 方向 | 說明 |
|---|---|
| 更多故障 AI 類型 | 販賣機、無人機、廣告招牌、AI 警察、自動廁所... |
| 更多收藏媒材 | 觸感類(盲文卡)、嗅覺類(舊香水)、味覺類(手寫食譜) |
| 副業 NPC 群 | 多位收藏家、雜誌主編、研究員,各有偏好與背景故事 |
| 道德軸累積 | 賣 vs 留 vs 還或發表,長期影響結局 |
| 兼職其他身份 | 偶爾接「非法資料搬運」或「肉身資料農夫」單,給玩家多樣性 |
| 收藏室視覺化 | 公寓變成玩家的個人博物館 / 牆面 / 文件夾 |
